# WorthMatch V4 Demo

独立于 `worthmatch_v2` 的前端原型，用于验证“AI 找搭子 + ABTI 动物搭子人格测试”的新方向。

## 打开方式

直接用浏览器打开：

```text
/Users/leo/Desktop/worthmatch/worthmatch_v4/index.html
```

这个 demo 不依赖后端服务，所有数据都是本地 mock。

## 已包含

- 今日：AI 输入需求，生成搭子邀请卡。
- 找搭子：按具体需求推荐搭子候选。
- ABTI：动物搭子人格测试和 16 类动物展示。
- AI：聊天式个人 Agent 和记忆面板。
- 发现：八类搭子场景入口。
- 社群：小群、小局、固定搭子队。
- 消息：聊天与 AI 破冰建议。
- 我的：照片、ABTI、认证、隐私边界。
